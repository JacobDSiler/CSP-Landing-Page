In Action
See the workflow
Object Mode
Core Workflow
Draw. Cut. Done.
Activate the tool, draw a line across your mesh, press Enter — and the cut is made. It really is that simple. Works on any mesh, any topology.
  • Works in Object, Edit, and Sculpt mode
  • Straight lines or smooth bezier curves
  • Trim, Slice, or Intersect with one keypress
  • Live red preview before committing the cut
Bezier Mode
Curve Control
Bezier curves? Easy.
Toggle bezier mode mid-draw to get perfectly smooth, organic cuts. Control handles let you dial in exactly the curve you need — no approximating with extra points.
  • Press B to toggle bezier on the fly
  • Drag handles to adjust curve tension
  • Ideal for organic shapes and character work
  • Same precision as straight-line mode
Closed Loop
Closed Loops
Punch holes with closed loops.
Close your curve to create a loop — CurveSlice Pro automatically detects whether you're cutting inward or outward and handles it correctly every time.
  • Close the curve to activate loop mode
  • Punch holes or extract regions
  • Inward/outward detection is automatic
  • Flip with F if needed
Auto Remesh
Clean topology after every cut
Choose your remesh method and CurveSlice Pro handles it automatically. From raw boolean topology to quad-dominant production meshes — without leaving Blender.
